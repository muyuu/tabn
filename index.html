  • htmlにタブを書く

    <div class="js-tab">
												<ul class="js-tab__head">
												<li class="js-tab__item">タブ1</li>
												<li class="js-tab__item">タブ2</li>
												<li class="js-tab__item">タブ3</li>
												</ul>
												<ul class="js-tab__body">
												<li class="js-tab__content">中身ですよ、中身。文句あっか</li>
												<li class="js-tab__content">中身ですよ、中身。文句あっか</li>
												<li class="js-tab__content">中身ですよ、中身。文句あっか</li>
												</ul>
												</div>

    css はお好みで。

    jQueryを読み込む

    Underscore.jsを読み込む

    <script src="//code.jquery.com/jquery-1.10.1.min.js"></script>
												<script src="//cdnjs.cloudflare.com/ajax/libs/underscore.js/1.7.0/underscore-min.js"></script>

    tabn.jsを読み込む

    <script src='/tab.js'></script>

    準備は以上になります！

  • タブとタブのコンテンツをラップした要素をセレクターに指定

    app.tab.set({".tab"});

    アクティブタブとコンテンツには .is-current が付与される

    タブはデフォルトはセレクタ内の .js-tab__item 。

    タブをラップする要素はデフォルトはセレクタ内の .js-tab 。変更したい場合はオプションを付与。

  • root タブの要素（デフォルトは .js-tab ）
    tab タブのラッパー（デフォルトは .js-tab__head ）
    item タブ（デフォルトは .js-tab__item ）
    body タブの中身のラッパー（デフォルトは js-tab__body ）
    content タブの中身（デフォルトは .js-tab__content ）
    currentClass カレントなタブとコンテンツに付与されるクラス（デフォルトは is-current ）
